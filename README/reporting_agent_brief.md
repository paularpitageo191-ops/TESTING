## 🧠 Core Functionalities

### 1. Healing Log Parsing
Reads latest healing log:
- Extracts PASS / FAIL
- Builds test list

---

### 2. Jira Validation
Calls:
GET /rest/api/2/myself

---

### 3. Failure Handling
For each failed test:
- Creates Bug
- Attaches screenshot
- Links Bug → Story
- Adds comment to Story

---

### 4. Success Handling
- Posts summary comment
- Attaches sample screenshot

---

### 5. Markdown Report Generation

Generates structured report:

- Execution Summary
- Test Results
- Analysis
- Failure Details (with screenshots)
- Business Impact
- Recommendations
- Jira Links
