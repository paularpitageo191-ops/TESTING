# DOM Capture Agent — Technical Brief

## 📁 File
dom_capture.py

## 🎯 Purpose
The DOM Capture Agent (Discovery Engine) extracts the live UI structure of a web application.

It captures:
- DOM elements (inputs, buttons, dropdowns, etc.)
- Form structures
- Accessibility tree
- Shadow DOM + iframes
- Screenshots

It enables downstream:
- Gherkin generation
- Step generation
- Self-healing
- UI vector memory

## 🧠 Core Functionalities

### 1. Navigation & Login
- Navigates to BASE_URL
- Performs heuristic login (multi-step aware)
- Supports session restore

---

### 2. Page Stabilization
- Waits for network idle
- Removes loaders
- Scrolls page for lazy content

---

### 3. DOM Extraction
Runs in-browser JS engine to extract:

- Inputs
- Buttons
- Dropdowns (native + custom)
- Textareas
- Interactive elements
- Labels
- Form structure

---

### 4. Advanced Coverage

- Shadow DOM traversal
- Iframe extraction (same-origin)
- Accessibility tree capture
- Intent classification (AI-ready)

---

### 5. Screenshot Capture
- Full page screenshots per URL

---

### 6. Vectorization (Optional)
- Converts elements → embeddings
- Uploads to Qdrant
- Enables semantic UI memory
